# 无线链路协议 v3

## 帧格式

所有多字节整数使用大端序。

| 偏移 | 长度 | 内容 |
| --- | --- | --- |
| 0 | 2 | 固定魔数 `44 53` |
| 2 | 1 | 协议版本 `03` |
| 3 | 1 | 帧类型 |
| 4 | 4 | 同步码派生的网络 ID |
| 8 | 4 | 本次启动会话 ID |
| 12 | 4 | 递增序号 |
| 16 | 1 | 负载长度 |
| 17 | 0..64 | 负载 |

接收端使用“会话 ID、序号、类型、长度、负载摘要”共同识别重复帧。
每次射频初始化还会可靠发送 `SESSION_START`。接收端无条件接受该帧并
清空上一会话的重复包缓存，因此协议正确性不依赖随机数绝对唯一；设备
重启后即使会话 ID 和序号碰撞，也不会复用上一次启动缓存的 SWD 结果。

## ACK 链路指标

ACK 负载固定为 7 字节：

| 偏移 | 内容 |
| --- | --- |
| 0..1 | 接收 RSSI，单位 0.5 dBm，有符号大端整数 |
| 2 | SX1281 packet error status |
| 3 | SX1281 TX/RX status |
| 4 | SX1281 sync address status |
| 5 | 接收端当前 profile |
| 6 | 接收端当前射频频道索引 |

发送端仅在 `MODE=AUTO` 时将 ACK RSSI 输入 EWMA 和速率决策。固定速率
模式只记录指标，不改变 profile。

## 自适应跳频

无线主机每累计 8 个成功业务帧发起一次频道切换：

1. 主机在旧频道可靠发送 `HOP_SWITCH`。
2. 从机在旧频道回复 ACK，发完后切到目标频道并开始 360 ms 试用。
3. 主机收到 ACK 后切换，并在新频道可靠发送 `HOP_CONFIRM`。
4. 从机确认后回复 ACK；确认事务失败时双方回退旧频道。

频道表为 2405 MHz 到 2480 MHz，共 16 个频道，间隔 5 MHz。同步码派生
频道排列与初始会合频道。普通业务帧重传会避开连续失败的频道；接收端
连续 180 ms 未收到有效帧后，以 35 ms 驻留时间扫描频道表，因此当前频道
或初始会合频道被持续占用时仍可恢复。详细设计见 `frequency_hopping.md`。

## Profile 切换

无线主机拥有切速控制权：

1. 主机使用旧 profile 可靠发送 `PROFILE_SWITCH`。
2. 从机使用旧 profile 回复带指标的 ACK。
3. 从机在 ACK 发完后切换到目标 profile，并启动 300 ms 试用窗口。
4. 主机收到 ACK 后切换，并立即可靠发送 `PROFILE_CONFIRM`。
5. 从机收到确认或其他目标 profile 下的有效业务帧后结束试用。
6. 如果确认未到达，从机在试用窗口超时后回退旧 profile。
7. 如果切换 ACK 丢失，主机会继续在旧 profile 重传；从机回退后重新
   接收该控制帧，并再次执行切换。
8. 如果目标 profile 下的确认重试全部耗尽，主机也回退到切换前 profile，
   两端不会分别落入不同的恢复速率。

无线恢复会重新使用配置中的初始 profile；自动模式初始值为 GFSK 1 Mbps。
AUTO 模式下任一端连续 500 ms 没有收到有效帧时，也会回到 GFSK 1 Mbps
会合 profile。这样单边重启、射频恢复或切速状态丢失后，两端可以重新
收到 `SESSION_START` 并恢复通信。

## CMSIS-DAP 诊断命令

`DAP Vendor 0x80` 状态格式版本为 5。原有字节 0..20 保持不变：

| 偏移 | 内容 |
| --- | --- |
| 21..22 | 对端 RSSI，0.5 dBm，有符号小端整数 |
| 23 | 当前 profile |
| 24 | 本次启动的 profile 切换次数 |
| 25 | 对端 packet error status |
| 26 | 对端 TX/RX status |
| 27 | 对端 sync address status |
| 28 | 当前射频频道索引 |
| 29 | 本次启动的频道切换次数 |
| 30 | 上次复位原因位图 |
| 31..34 | 本次启动运行时间，单位 ms |
| 35..38 | RX/TX timeout IRQ 次数 |
| 39..42 | 协议校验失败帧数 |
| 43..46 | 对端 session 变更次数 |

状态标志字节 bit3 表示已经收到有效的对端链路指标。
